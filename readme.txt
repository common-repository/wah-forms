Simple and powerful contact form builder.

== Description ==
Build contact form with drag and drop functionality. All submissions saves to Leads, you can access all submissions any time. You can customize your forms, upload files and save leads.

== PRIVACY NOTICES ==
* WAHForm do NOT send any user data to external services.
* WAHForm do NOT use cookies.

== Installation ==
Default WordPress plugin installation.

== Frequently Asked Questions ==

= How to see form submissions? =
You can see all submissions under "Leads" tab in the form admin panel.

= How to display form on page/post? =
Simple! Paste the form shortcode inside your page. The shortcode located at the top left corner of the WAHForms settings screen.

= How can I report security bugs? =
You can report security bugs through the Patchstack Vulnerability Disclosure Program. The Patchstack team help validate, triage and handle any security vulnerabilities. [Report a security vulnerability.](https://patchstack.com/database/vdp/wah-forms)
